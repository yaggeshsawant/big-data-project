"FMCG Global Sales Data Pipeline” 				
A fully automated, multi-source FMCG sales data pipeline in which 1.1 Million Sales transactions from 7 different countries are streamed, cleaned, transformed, and stored across cloud platforms end-to-end, with zero manual intervention.
Tech Stack:
• Apache Kafka: for real-time batch streaming of sales transaction data.
• HDFS: for distributed storage of streamed data in Parquet format.
• AWS + Snowflake + Google Drive: as multi-source data inputs (store, product & sales data).
• Apache Hive: for structured storage of raw and processed datasets.
• PySpark: for large-scale data cleaning, preprocessing & transformation.
• Apache Airflow: for orchestrating and automating the entire pipeline workflow.
• AWS: for final storage of processed data, ready for reporting & consumption.
• Tableau: for data visualization, presenting insightful and interactive dashboards from the processed data.
